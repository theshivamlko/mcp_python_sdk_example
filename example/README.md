

### MCP Inspector

Run STDIO MCP Server  </br>
`uv run mcp dev server.py:notes_mcp`


Run HTTP MCP Server `http://127.0.0.1:7000/mcp`  </br>
`uv run server.py`

now run MCP Client </br>
`uv run mcp_client.py`


API keys for PriceScout in `.env`
```
OPENAI_KEY=
FIRECRAWL_API_KEY=
ANTHROPIC_API_KEY=
```
